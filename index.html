Django CRUD Produits

Projet de gestion de produits en architecture MVT

Cette page présente le projet généré. L’application principale est construite avec Django, des templates séparés, un fichier JavaScript dédié pour le frontend et une structure organisée par dossiers.

Fonctionnalités

  • Création de produits
  • Liste et recherche de produits
  • Modification de produits
  • Suppression de produits
  • Templates Django et JavaScript simple

Commandes de lancement

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
